# lispo
Standard Bourne-like, POSIX-compliant shell functions library for developers.

#### INSTALATION:

Just download or clone it and link .inc files from your shell script like this:
```bash
. /path/file.inc
```
Recommended path for .inc files is /usr/lib/sh/
